# 🛍️ FreshCart – E-commerce Web Application

A modern e-commerce web application built with **Next.js** and **TypeScript**.

## 🌟 Overview
FreshCart is a scalable online storefront developed with Next.js. It includes routing, product pages, a responsive layout, and a foundation for adding shopping cart features and backend integration.

## 🛠️ Tech Stack
- **Next.js** (React framework)  
- **TypeScript**  
- **CSS / Tailwind / any CSS framework used**  
- **JavaScript**

## 📦 Features
- 🚀 Built with Next.js for fast performance and SSR  
- 📄 Clean and modular component structure  
- 📱 Fully responsive UI  
- 🧠 Designed for scalability and extensibility
